How does it work?

We offer short crash courses on selected topics embeded in an experimental setup. It is a great opportunity to learn new things quickly but also to contribute to personalized education.

Data gathered from these quick learning sessions allows us to improve learning experience in on-line education.

For teachers & researchers

Design an experiment using orchestration graphs. Track the results in the dashboard or export them and analyze using our R library.
